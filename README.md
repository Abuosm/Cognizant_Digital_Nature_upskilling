## ⚙️ Getting Started

### Prerequisites

* JDK 17+
* Apache Maven
* Oracle DB / SQL Developer
* IntelliJ IDEA / Eclipse / VS Code

### Clone Repository

```bash
git clone https://github.com/Abuosm/Cognizant_Digital_Nurture_upskilling.git
```

### Navigate to Project

```bash
cd Cognizant_Digital_Nurture_upskilling/LibraryManagement/LibraryManagement
```

### Build Project

```bash
mvn clean install
```
